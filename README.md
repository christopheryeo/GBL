# GBL

This is the main repository for the GBL project.

## Project Structure

```
GBL/
├── README.md
├── requirements.txt
└── src/
    └── main.py
```

## Getting Started

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python src/main.py
```
